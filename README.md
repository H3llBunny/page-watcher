## Options
Open **Options** from the popup.

- **Keywords**: comma‑separated (e.g., `1 - Critical, 2 - High`). You can add P3 and P4 if you want.
- **Refresh interval (seconds)**: must be **≥ 60** (Chrome MV3 alarms minimum).
- **Play sound**: on/off.
- **Desktop notification**: on/off.
- **Keep computer awake while scanning**: on/off.
---
Install / Load in Chromium browser (Developer mode)
1. Go to **Manage extensions**.
2. Toggle **Developer mode**.
3. Click **Load unpacked** and select the repo folder.
4. Pin the extension (optional), open the popup → **Start**.
